## 🎯 ¿Qué es BAS?

BAS (Business Automation System) es una plataforma SaaS diseñada para **democratizar la automatización digital** para negocios pequeños y medianos.

**Resuelve el problema:** PYMEs sin tiempo, conocimientos ni presupuesto para tener presencia online competitiva.

---

## ✨ Features

### 🌐 **Módulo 1: Presencia Digital**
- Landing page profesional personalizada
- SEO optimizado automáticamente
- Mobile-first responsive
- Setup en 24 horas

### 🎯 **Módulo 2: Captura de Leads**
- QR para negocios físicos
- Formularios inteligentes
- WhatsApp Business integrado
- Base de datos automática

### ✍️ **Módulo 3: Generador de Contenido**
- Posts para redes sociales auto-generados
- Calendario editorial 4 semanas
- Hashtags inteligentes por nicho
- Scripts de Reels/Stories

### 📊 **Módulo 4: Mini-CRM**
- Gestión de clientes simple
- Historial de interacciones
- Recordatorios automáticos
- Segmentación inteligente

### 📧 **Módulo 5: Marketing Automatizado**
- Email sequences pre-configuradas
- Ofertas programadas
- Follow-ups sin intervención
- Cumpleaños/aniversarios

### 🛒 **Módulo 6: Tienda Online**
- Catálogo fácil de gestionar
- Checkout por WhatsApp/Bizum
- Sin complicaciones técnicas
- Integración redes sociales

## 🎨 Tech Stack

### Frontend
- HTML5 / CSS3 / JavaScript vanilla
- Mobile-first responsive design
- Intersection Observer API (animaciones)
- CSS Variables para theming

### Backend (próximamente)
- Node.js + Express
- PostgreSQL
- JWT authentication
- RESTful API

### Integraciones
- WhatsApp Business API
- Google Sheets / Airtable
- Zapier / Make

## 🚀 Quick Start

### Ver localmente

```bash
cd bas-saas
python3 -m http.server 8000
```

Abre: http://localhost:8000

### Deploy a GitHub Pages

1. Crear repo en GitHub
2. Push código:
```bash
git remote add origin https://github.com/tu-usuario/bas-saas.git
git push -u origin master
```
3. Activar Pages en Settings → Pages
4. Listo: `https://tu-usuario.github.io/bas-saas/`

Ver [DEPLOY.md](DEPLOY.md) para más opciones.

## 📊 Business Model
