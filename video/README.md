## 🚀 OPCIÓN RECOMENDADA: Synthesia

### Paso 1: Crear cuenta
- Ve a: https://www.synthesia.io/
- Prueba gratis o plan básico ($30/mes)

### Paso 2: Crear nuevo video
1. Click "Create video"
2. Selecciona "Blank template"
3. Duración: 3 minutos

### Paso 3: Pegar guion
1. Abre `synthesia-script.txt`
2. Copia TODO el contenido
3. Pégalo en el editor de Synthesia
4. Selecciona avatar: "Masculino profesional" (ej: "James")
5. Idioma: Spanish (Spain o Latin America)

### Paso 4: Añadir screenshots
Captura estas pantallas del sitio (https://rodrigodiazv.github.io/BAS-SAAS/):

**Login primero:**
- Email: admin@bas-saas.com
- Password: admin123

**Screenshots necesarios:**

1. **Landing page** (0:00-0:20)
   - Hero section completa
   - Botón "Únete al grupo VIP" visible

2. **Dashboard** (0:20-1:00)
   - Vista completa con las 4 métricas
   - Gráficos de ingresos y usuarios
   - Captura en resolución 1920x1080

3. **Usuarios** (1:00-1:40)
   - Tabla de usuarios con filtros
   - Modal de edición abierto

4. **Suscripciones** (1:40-2:20)
   - Plan actual destacado
   - Grid de todos los planes
   - Historial de pagos

5. **Configuración** (2:20-2:50)
   - Sección de perfil
   - Toggles de notificaciones
   - Botón "Exportar datos"

6. **Cierre** (2:50-3:00)
   - Landing con botón Telegram
   - QR code del grupo (opcional)

### Paso 5: Sincronizar
- Arrastra cada screenshot al momento correcto del timeline
- Ajusta duración para que coincida con narración

### Paso 6: Música de fondo
- Selecciona música "Corporate" o "Tech"
- Volumen: 20-30%

### Paso 7: Generar
- Click "Generate video"
- Tiempo de procesamiento: 10-15 minutos
- Descarga en formato MP4

## 📸 Cómo capturar screenshots profesionales

### Herramientas:
- **Mac:** Cmd+Shift+4
- **Windows:** Windows+Shift+S
- **Linux:** Flameshot / GNOME Screenshot
- **Navegador:** F12 → Device toolbar → 1920x1080

### Configuración:
1. Resolución navegador: 1920x1080 (100% zoom)
2. Modo incógnito (UI limpia)
3. Sin extensiones visibles
4. Scroll suave para capturas largas

### Formato:
- PNG (sin compresión)
- 1920x1080 píxeles
- Nombre: `scene-01-landing.png`, `scene-02-dashboard.png`, etc.

## 💡 Tips profesionales

1. **Pronunciación:** Revisa que "BAS" se diga como "Bas" no "B-A-S"
2. **Ritmo:** 140 palabras/minuto es perfecto (ni muy rápido ni lento)
3. **Pausas:** 1 segundo entre escenas para respirar
4. **Transiciones:** Fade suave de 0.5s (no cortes bruscos)
5. **Música:** Que no compita con la voz (20-30% volumen)
6. **Miniatura:** Captura del dashboard con texto "Demo 3 min"
